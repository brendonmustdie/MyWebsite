What is a Broswer?

Week 4

According to Brittanica, browsers are software that interpret html to display data and allow computer users to view information on the internet. According to Mozilla, a web browser retrieves information from parts of the web to take you anywhere on the internet – the information is conveyed using the “hypertext transfer protocol” which defines how information is transmitted on the web.

When the browser retrieves information, it uses a rendering engine that translates the date into text and images – the date is written in HTML and browsers convert this code into what we see, hear, and experience.
